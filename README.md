# Portfolio 

Template de portfolio moderne en **HTML + CSS + JavaScript + PHP**.

## Fichiers

- `index.html` : page principale
- `style.css` : design responsive
- `script.js` : animations, theme toggle, envoi AJAX
- `contact.php` : traitement du formulaire de contact

## Lancer en local (PHP)

Prerequis : PHP installe.

```bash
php -S localhost:8000
```

Puis ouvre [http://localhost:8000](http://localhost:8000)

## A personnaliser

- Nom/logo dans `index.html`
- Tes vrais projets (titres, descriptions, liens)
- Adresse email dans `contact.php` (`$to`)
